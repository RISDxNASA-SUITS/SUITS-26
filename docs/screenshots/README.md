# Screenshots (optional)

Add static images here for README, presentations, or handoff. Not required for the app to run.

Suggested filenames (examples):

| File | Content |
|------|---------|
| `console-overview.png` | Full EVA AIA mission console (command + assistant + side panels) |
| `alerts-warnings.png` | Alerts panel with at least one active warning |
| `procedure-active.png` | Procedure panel mid-EGRESS_UIA with step text |
| `guardrail-reject.png` | Assistant panel showing a rejected command + `error_code` |

**How to capture**

1. Run backend (`uvicorn`) and frontend (`npm run dev`).
2. Use browser zoom ~100%, clear personal data from the shot if needed.
3. Save PNG or WebP into this folder.
4. Reference from the root `README.md` if desired: `docs/screenshots/console-overview.png`.
